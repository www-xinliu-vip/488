伍佰（Wu Bai），本名吴俊霖，1968年1月14日出生于中国台湾省新北市新店区，中国台湾男歌手、音乐人、吉他手、演员、摄影家。1990年以歌曲《小人国》正式出道。1992年参与电影《少年吔，安啦》原声带的词曲创作及演唱，发行首张个人专辑《爱上别人是快乐的事》；同年组建摇滚乐团China Blue，并担任主唱兼主音吉他手。1995年凭借歌曲《浪人情歌》和现场专辑《伍佰的Live》获得关注。
